Quick Start

Up & Running in 2 Minutes

1

Install & Configure

Clone the repo and set up your API keys.

git clone https://github.com/ebwood/LOLRoaster.git
cd LOLRoaster
npm install
cp .env.example .env
# Edit .env with your Gemini API key
2

Launch

Start the app and open the dashboard.

npm run dev
# Open http://localhost:8099
3

Play League

Start a game. The coach will auto-detect your client and begin roasting. Enjoy (or suffer)! üî•
